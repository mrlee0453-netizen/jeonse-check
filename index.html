    🗺 카카오맵 API 설정 가이드

    무료 API 키 발급 후 index.html에 한 줄만 수정하면 됩니다

    1

    카카오 개발자 계정 만들기

    카카오 계정이 있으면 바로 사용 가능합니다.
    developers.kakao.com → 로그인 → 내 애플리케이션

    2

    애플리케이션 등록

    애플리케이션 추가하기 → 앱 이름: 안심전세체크 → 저장
    생성된 앱의 JavaScript 키를 복사합니다.

    3

    플랫폼 등록 (GitHub Pages URL 입력)

    앱 설정 → 플랫폼 → Web → 사이트 도메인 추가:
    GitHub Pages 배포 시: https://[내 깃허브 아이디].github.io

    예시: https://honggildong.github.io
    4

    index.html에서 API 키 교체

    index.html 파일을 열어 아래 줄을 찾아 본인의 키로 교체하세요:

    // ↓↓ 이 줄을 찾아 YOUR_KAKAO_API_KEY를 실제 키로 교체 const KAKAO_API_KEY = 'YOUR_KAKAO_API_KEY';
    5

    GitHub Pages 배포하기

    1. GitHub에서 새 레포지토리 생성 (이름: jeonse-check)
    2. index.html 파일 업로드
    3. Settings → Pages → Source: main branch 선택
    4. 몇 분 후 https://[아이디].github.io/jeonse-check 접속 가능!

    배포 URL 예시: https://honggildong.github.io/jeonse-check
